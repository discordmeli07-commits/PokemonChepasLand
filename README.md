## 📌 Descripción

**ChepasLand** es un *fan game* inspirado en Pokémon de tercera generación.  
Incluye:

- 🏘️ Tres pueblos: **Pueblo Smix**, **ChepasLand**, **Asix**
- ⚔️ Combate por turnos estilo Pokémon
- 😆 Pokémon con **caras de compañeros de clase**
- 🏆 Dos gimnasios
- 🗺️ Mapas retro estilo GBA
- 🎨 Sprites personalizados y recursos libres
